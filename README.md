# Aashir Siddiqui — Personal Portfolio 2026

A modern, performant personal portfolio built with **React 19 + Vite**, featuring automated GitHub project syncing, 3D visuals, and a fully functional contact form.

## Features

- **Loading Screen** — Animated intro with session-based skip
- **Hero Section** — Typed-text effect, CTA buttons, Three.js interactive 3D scene
- **About Section** — Bio, mission, strengths, and values
- **Skills Section** — Categorized technology cards with icons
- **Projects Section** — Auto-synced from GitHub, filter/search/3D tilt cards
- **GitHub Dashboard** — Stats, top languages, contribution heatmap
- **Contact Form** — Serverless function sending email via Resend
- **SEO** — SSG prerendering, OG tags, Twitter cards, JSON-LD schema
- **Responsive** — Fully usable from 375px to 4K
- **Dark Theme** — Premium glassmorphism aesthetic (#050816 base)

## Tech Stack

| Layer        | Technology                                      |
| ------------ | ----------------------------------------------- |
| Framework    | React 19 + Vite 8                               |
| Styling      | Tailwind CSS 4                                  |
| Animation    | Framer Motion                                   |
| 3D           | React Three Fiber + Drei + Three.js             |
| Icons        | React Icons                                     |
| SSG/SEO      | vite-react-ssg                                  |
| Hosting      | Vercel (auto-deploy on push)                    |
| CI/CD        | GitHub Actions (scheduled project sync)         |
| Contact      | Vercel Serverless Function + Resend API         |
| Project Data | Static JSON (generated by CI, never client API) |

## Installation

```bash
git clone https://github.com/aashirsohail104/portfolio.git
cd portfolio
npm install
```

## Development

```bash

```

## Environment Variables

| Variable              | Where        | Purpose                         |
| --------------------- | ------------ | ------------------------------- |
| `RESEND_API_KEY`      | Vercel       | Sends contact form emails       |
| `CONTACT_TO_EMAIL`    | Vercel       | Where form submissions go       |
| `SITE_URL`            | Vercel       | Used for OG tags / canonical    |
| `PORTFOLIO_SYNC_TOKEN`| GitHub Secrets | GitHub API token for project sync |
